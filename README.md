# ATUA Assignment 1: Spatial Analysis of Airbnb Prices

This repository contains Assignment 1 for ATUA, which investigates the spatial pattern of Airbnb prices and compares global and local spatial regression models.

## Contents
- data exploration and preprocessing
- KNN spatial weight matrix construction
- OLS, SEM, SAR, and SDM estimation
- GWR and MGWR modelling
- residual spatial autocorrelation testing with Moran’s I
- comparison of spatial model performance

## Main files
- `ATUA_Assignment_1.ipynb`
- `3149531_Assignment_1.html`
- `README.md`

## Summary
The analysis shows that Airbnb prices exhibit significant spatial dependence. Compared with OLS, spatial regression models better account for spatial structure, and SDM performs best among the global models. GWR and MGWR further reveal spatial heterogeneity in the effects of explanatory variables. :contentReference[oaicite:2]{index=2}
